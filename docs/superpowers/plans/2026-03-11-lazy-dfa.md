# Lazy DFA Engine Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a lazy/hybrid DFA engine that builds DFA states on demand during search, providing O(1) per-character throughput for patterns without look-assertions.

**Architecture:** The lazy DFA sits between the NFA and the meta engine's `Strategy.Core`. It receives a compiled NFA and a `CharClasses` equivalence map. During search, it lazily constructs DFA states via powerset/epsilon closure, caching transitions in a flat `int[]` table indexed by pre-multiplied state IDs. `Strategy.Core` tries the lazy DFA first; on `GaveUp` or for patterns with look-assertions, it falls back to PikeVM. A two-phase approach finds match end (forward DFA) then match start + captures (PikeVM on narrowed window).

**Tech Stack:** Java 21, JUnit 5, Maven (./mvnw wrapper)

**Spec:** `docs/superpowers/specs/2026-03-11-lazy-dfa-design.md`

**Context:** Char equivalence classes partition the 65,536 UTF-16 char values into groups that behave identically across all NFA transitions. For example, in the pattern `[a-z]+`, chars `a`-`z` are one class, and everything else is another. This reduces the DFA's alphabet size from 65,536 to ~30-80 classes, making transition tables small.

The upstream Rust crate uses `ByteClasses` (`upstream/regex/regex-automata/src/util/alphabet.rs`) which maps 256 byte values. We need 65,536 char values, so we use a two-level table: a 256-entry high-byte index pointing to shared 256-entry rows. Identical rows are deduplicated. See spec section "Char Equivalence Classes".

```java
package lol.ohai.regex.automata.dfa;

/**
 * Maps each UTF-16 char to an equivalence class ID.
 *
 * <p>Two chars belong to the same equivalence class iff they are treated
 * identically by every transition in the NFA. This reduces the DFA alphabet
 * from 65,536 to typically 30-80 classes.</p>
 *
 * <p>Uses a two-level table: {@code highIndex[c >>> 8]} selects a row,
 * then {@code rows[row][c & 0xFF]} gives the class ID. Identical rows
 * are shared, keeping memory ~1-5KB for typical patterns.</p>
 */
public final class CharClasses {
    private final byte[][] rows;
    private final int[] highIndex;
    private final int classCount;
    private final int stride;
    private final int strideShift;

    CharClasses(byte[][] rows, int[] highIndex, int classCount) {
        this.rows = rows;
        this.highIndex = highIndex;
        this.classCount = classCount;
        // stride = next power of 2 >= classCount + 1 (for EOI)
        int alphabetSize = classCount + 1; // +1 for EOI
        this.stride = Integer.highestOneBit(alphabetSize - 1) << 1;
        this.strideShift = Integer.numberOfTrailingZeros(this.stride);
    }

    /** Returns the equivalence class ID for the given char. O(1). */
    public int classify(char c) {
        return rows[highIndex[c >>> 8]][c & 0xFF] & 0xFF;
    }

    /** Number of char equivalence classes (not counting EOI). */
    public int classCount() { return classCount; }

    /** The EOI (end-of-input) class ID, equal to classCount. */
    public int eoiClass() { return classCount; }

    /** Stride for transition table rows: next power of 2 >= classCount + 1. */
    public int stride() { return stride; }

    /** log2(stride), for bit-shift operations. */
    public int strideShift() { return strideShift; }

    /**
     * Creates an identity mapping: all chars map to class 0.
     * Useful for testing and as a degenerate case.
     */
    public static CharClasses identity() {
        byte[] singleRow = new byte[256]; // all zeros
        byte[][] rows = { singleRow };
        int[] highIndex = new int[256]; // all point to row 0
        return new CharClasses(rows, highIndex, 1);
    }
}
```

**Context:** The builder walks all NFA states, collects char range boundaries from `CharRange` and `Sparse` transitions, then partitions the char space so that all chars within a partition behave identically across all transitions. The algorithm is:

1. Collect all range boundary chars from the NFA (every `start` and `end+1` from `CharRange` and `Sparse.Transition`)
2. Sort and deduplicate the boundary set
3. Each interval between consecutive boundaries is one equivalence class
4. Build the two-level table mapping each char to its class ID, with row deduplication

```java
package lol.ohai.regex.automata.dfa;

import lol.ohai.regex.automata.nfa.thompson.NFA;
import lol.ohai.regex.automata.nfa.thompson.State;
import lol.ohai.regex.automata.nfa.thompson.Transition;

import java.util.*;

/**
 * Builds {@link CharClasses} from an NFA's transitions.
 *
 * <p>Algorithm: collect all char range boundaries from the NFA, sort them,
 * then assign each interval between boundaries a distinct equivalence class.
 * Finally, build a two-level lookup table with row deduplication.</p>
 */
public final class CharClassBuilder {

    private CharClassBuilder() {} // utility class

    /**
     * Build char equivalence classes from the given NFA.
     *
     * @param nfa the compiled NFA
     * @return the equivalence class mapping
     */
    public static CharClasses build(NFA nfa) {
        // Step 1: Collect all range boundaries
        TreeSet<Integer> boundaries = new TreeSet<>();
        boundaries.add(0);           // start of char space
        boundaries.add(0x10000);     // end of char space (exclusive)

        for (int i = 0; i < nfa.stateCount(); i++) {
            State state = nfa.state(i);
            switch (state) {
                case State.CharRange cr -> {
                    boundaries.add(cr.start());
                    boundaries.add(cr.end() + 1);
                }
                case State.Sparse sp -> {
                    for (Transition t : sp.transitions()) {
                        boundaries.add(t.start());
                        boundaries.add(t.end() + 1);
                    }
                }
                default -> { /* no char transitions */ }
            }
        }

        // Step 2: Build flat char→class mapping
        // Each interval [boundaries[i], boundaries[i+1]) gets a class ID
        int[] sortedBounds = boundaries.stream().mapToInt(Integer::intValue).toArray();
        int classCount = sortedBounds.length - 1;

        // Class IDs are stored as unsigned bytes (0-255). If we have more than
        // 256 classes, the byte storage would overflow. In practice this requires
        // a pattern with 256+ disjoint char ranges, which is extremely rare.
        // Cap at 256 by merging excess intervals into the last class.
        if (classCount > 256) {
            classCount = 256;
        }

        // Build flat char→class map (temporary, 64KB)
        byte[] flatMap = new byte[65536];
        for (int cls = 0; cls < sortedBounds.length - 1; cls++) {
            int from = sortedBounds[cls];
            int to = Math.min(sortedBounds[cls + 1], 65536);
            int mappedClass = Math.min(cls, classCount - 1); // cap at max class
            for (int c = from; c < to; c++) {
                flatMap[c] = (byte) mappedClass;
            }
        }

        // Step 3: Build two-level table with row deduplication
        Map<ByteArrayKey, Integer> rowIndex = new HashMap<>();
        List<byte[]> uniqueRows = new ArrayList<>();
        int[] highIndex = new int[256];

        for (int hi = 0; hi < 256; hi++) {
            byte[] row = new byte[256];
            System.arraycopy(flatMap, hi * 256, row, 0, 256);
            ByteArrayKey key = new ByteArrayKey(row);
            Integer existing = rowIndex.get(key);
            if (existing != null) {
                highIndex[hi] = existing;
            } else {
                int idx = uniqueRows.size();
                uniqueRows.add(row);
                rowIndex.put(key, idx);
                highIndex[hi] = idx;
            }
        }

        return new CharClasses(
                uniqueRows.toArray(byte[][]::new),
                highIndex,
                classCount);
    }

    /** Wrapper for byte[] to use as HashMap key with content-based equals/hashCode. */
    private record ByteArrayKey(byte[] data) {
        @Override
        public boolean equals(Object o) {
            return o instanceof ByteArrayKey k && Arrays.equals(data, k.data);
        }

        @Override
        public int hashCode() {
            return Arrays.hashCode(data);
        }
    }
}
```

```java
package lol.ohai.regex.automata.dfa.lazy;

import java.util.Arrays;

/**
 * The content of a DFA state: the set of simultaneously active NFA states.
 *
 * <p>Two DFA states with the same NFA state set (and match flag) are considered
 * identical — they share a single transition table row. This deduplication is
 * the key to the lazy DFA's efficiency.</p>
 *
 * <p>The NFA state array must be sorted for consistent equality comparison.</p>
 */
public final class StateContent {
    private final int[] nfaStates;
    private final boolean isMatch;
    private final int hashCode;

    public StateContent(int[] nfaStates, boolean isMatch) {
        this.nfaStates = nfaStates;
        this.isMatch = isMatch;
        this.hashCode = Arrays.hashCode(nfaStates) * 31 + Boolean.hashCode(isMatch);
    }

    /**
     * Returns the sorted NFA state IDs. MUST NOT be mutated — this array
     * is used for equals/hashCode and is shared with the deduplication map.
     */
    public int[] nfaStates() { return nfaStates; }
    public boolean isMatch() { return isMatch; }

    @Override
    public boolean equals(Object o) {
        if (this == o) return true;
        if (!(o instanceof StateContent sc)) return false;
        return isMatch == sc.isMatch && Arrays.equals(nfaStates, sc.nfaStates);
    }

    @Override
    public int hashCode() { return hashCode; }
}
```

### Task 5: DFACache — Per-Search Mutable State

**Files:**
- Create: `regex-automata/src/main/java/lol/ohai/regex/automata/dfa/lazy/DFACache.java`
- Test: `regex-automata/src/test/java/lol/ohai/regex/automata/dfa/lazy/DFACacheTest.java`

**Context:** The `DFACache` holds all mutable state for a lazy DFA search: the transition table (flat `int[]`), state storage, deduplication map, start state cache, reusable workspace for epsilon closure, and give-up tracking counters. It's allocated once per thread (via `ThreadLocal<Strategy.Cache>` in `Regex.java`) and reused across searches.

Key constants from the spec:
- `UNKNOWN = 0` — transition not yet computed
- `DEAD = stride` — no match possible
- `QUIT = stride × 2` — gave up, fall back
- Match states have bit 31 set on their ID

The cache pre-allocates 3 sentinel state rows (UNKNOWN, DEAD, QUIT) during construction.

```java
package lol.ohai.regex.automata.dfa.lazy;

import lol.ohai.regex.automata.dfa.CharClasses;
import lol.ohai.regex.automata.util.SparseSet;

import java.util.*;

/**
 * Per-search mutable state for the lazy DFA.
 *
 * <p>Holds the transition table, state storage, deduplication map, and
 * give-up tracking. Not thread-safe — managed via {@code ThreadLocal}.</p>
 *
 * <p>The transition table is a flat {@code int[]} indexed by
 * {@code stateId + classId}. State IDs are pre-multiplied by stride,
 * so no multiplication is needed in the hot loop.</p>
 *
 * <p>Three sentinel states are pre-allocated at construction:</p>
 * <ul>
 *   <li>{@code UNKNOWN (0)} — transition not yet computed</li>
 *   <li>{@code DEAD (stride)} — no match possible from this state</li>
 *   <li>{@code QUIT (stride*2)} — gave up, fall back to PikeVM</li>
 * </ul>
 *
 * <p>Match states have bit 31 set on their ID. A {@code sid < 0} check
 * detects matches in the hot loop.</p>
 */
public final class DFACache {

    /** Sentinel: transition not yet computed. Always 0. */
    public static final int UNKNOWN = 0;

    /** Match flag: bit 31. */
    public static final int MATCH_FLAG = 0x8000_0000;

    /** Returns the DEAD sentinel ID for the given stride. */
    public static int dead(int stride) { return stride; }

    /** Returns the QUIT sentinel ID for the given stride. */
    public static int quit(int stride) { return stride * 2; }

    private final int stride;
    private int[] transTable;
    private final List<StateContent> states;
    private final Map<StateContent, Integer> stateMap;

    // Start state cache
    int startUnanchored = UNKNOWN;
    int startAnchored = UNKNOWN;

    // Reusable workspace for epsilon closure
    final SparseSet nfaStateSet;
    int[] closureStack;  // may grow during epsilon closure

    // Give-up tracking
    private int clearCount;
    long charsSearched;
    int statesCreated;

    private final int maxStates;

    /**
     * Creates a new cache.
     *
     * @param charClasses the equivalence class map (for stride)
     * @param capacityBytes approximate memory budget for the transition table
     */
    public DFACache(CharClasses charClasses, int capacityBytes) {
        this(charClasses, capacityBytes, 0);
    }

    /**
     * Creates a new cache.
     *
     * @param charClasses the equivalence class map (for stride)
     * @param capacityBytes approximate memory budget for the transition table
     * @param nfaStateCount number of NFA states (for workspace sizing)
     */
    public DFACache(CharClasses charClasses, int capacityBytes, int nfaStateCount) {
        this.stride = charClasses.stride();
        int bytesPerState = stride * 4; // 4 bytes per int
        this.maxStates = Math.max(4, capacityBytes / bytesPerState);

        // Initial transition table: enough for sentinels + some user states
        int initialCapacity = Math.min(maxStates, 64);
        this.transTable = new int[initialCapacity * stride];
        this.states = new ArrayList<>();
        this.stateMap = new HashMap<>();

        // Workspace
        this.nfaStateSet = new SparseSet(Math.max(1, nfaStateCount));
        this.closureStack = new int[Math.max(1, nfaStateCount) * 2];

        // Initialize sentinel states
        initSentinels();
    }

    private void initSentinels() {
        // UNKNOWN (ID=0): all transitions are UNKNOWN (already 0 in Java)
        states.add(new StateContent(new int[0], false));

        // DEAD (ID=stride): all transitions are DEAD
        int deadId = dead(stride);
        ensureCapacity(deadId + stride);
        Arrays.fill(transTable, deadId, deadId + stride, deadId);
        states.add(new StateContent(new int[0], false));

        // QUIT (ID=stride*2): all transitions are QUIT
        int quitId = quit(stride);
        ensureCapacity(quitId + stride);
        Arrays.fill(transTable, quitId, quitId + stride, quitId);
        states.add(new StateContent(new int[0], false));
    }

    /**
     * Returns the next state ID for the given current state and class.
     * @param stateId raw (un-flagged) state ID — must NOT have MATCH_FLAG set
     */
    public int nextState(int stateId, int classId) {
        return transTable[stateId + classId];
    }

    /**
     * Sets a transition in the table.
     * @param stateId raw (un-flagged) state ID — must NOT have MATCH_FLAG set
     */
    public void setTransition(int stateId, int classId, int nextStateId) {
        transTable[stateId + classId] = nextStateId;
    }

    /**
     * Allocates a new DFA state or returns an existing one if the content
     * has been seen before.
     *
     * @param content the NFA state set and match flag
     * @return the state ID (pre-multiplied by stride, with MATCH flag if applicable)
     */
    public int allocateState(StateContent content) {
        Integer existing = stateMap.get(content);
        if (existing != null) return existing;

        int rawId = states.size() * stride;
        int sid = content.isMatch() ? (rawId | MATCH_FLAG) : rawId;

        ensureCapacity(rawId + stride);
        // New state's transitions default to UNKNOWN (already 0)

        states.add(content);
        stateMap.put(content, sid);
        statesCreated++;
        return sid;
    }

    /** Returns the StateContent for a raw (un-flagged) state ID. */
    public StateContent getState(int rawStateId) {
        return states.get(rawStateId / stride);
    }

    /** Number of allocated states (including sentinels). */
    public int stateCount() { return states.size(); }

    /** Whether the cache is full and needs clearing. */
    public boolean isFull() { return states.size() >= maxStates; }

    /** Number of times the cache has been cleared. */
    public int clearCount() { return clearCount; }

    /** Stride value. */
    public int stride() { return stride; }

    /**
     * Clears all states except sentinels. Optionally preserves one state
     * (the current state during a search) by re-adding it.
     *
     * @param preserve state content to re-add after clearing, or null
     */
    public void clear(StateContent preserve) {
        // Reset transition table
        Arrays.fill(transTable, 0);
        states.clear();
        stateMap.clear();

        // Re-init sentinels
        initSentinels();

        // Re-add preserved state if provided
        if (preserve != null) {
            allocateState(preserve);
        }

        // Reset start state cache
        startUnanchored = UNKNOWN;
        startAnchored = UNKNOWN;

        clearCount++;
        charsSearched = 0;
        statesCreated = 0;
    }

    private void ensureCapacity(int needed) {
        if (needed > transTable.length) {
            int newSize = Math.max(transTable.length * 2, needed);
            transTable = Arrays.copyOf(transTable, newSize);
        }
    }
}
```

**Context:** This is the main engine. `LazyDFA` is immutable and thread-safe. It holds references to the NFA and `CharClasses`. The `searchFwd()` method implements the forward search loop from the spec.

Key behaviors:
1. **Look-assertion bailout:** `LazyDFA.create()` scans the NFA for `State.Look` states. If found, returns `null`.
2. **Epsilon closure:** Stack-based DFS through Union, BinaryUnion, Capture states. Only `CharRange`, `Sparse`, and `Match` are added to the result set. `Look` states should not appear (we bailed out), but if encountered, give up.
3. **Determinization:** When a transition is `UNKNOWN`, compute the next DFA state via powerset construction: get source state's NFA states → follow char transitions → epsilon closure → `StateContent` → allocate or deduplicate.
4. **Search loop:** The hot path is a single array lookup. Match detection via `sid < 0`. Sentinel handling for UNKNOWN, DEAD, QUIT.
5. **Cache clearing + give-up:** When cache is full, clear and retry. After `MIN_CLEAR_COUNT` (3) clears, check `charsSearched / statesCreated < MIN_CHARS_PER_STATE` (10) → give up.

```java
package lol.ohai.regex.automata.dfa.lazy;

import lol.ohai.regex.automata.dfa.CharClasses;
import lol.ohai.regex.automata.nfa.thompson.NFA;
import lol.ohai.regex.automata.nfa.thompson.State;
import lol.ohai.regex.automata.nfa.thompson.Transition;
import lol.ohai.regex.automata.util.Input;

import java.util.Arrays;

/**
 * A lazy (hybrid) DFA that builds states on demand during search.
 *
 * <p>Immutable and thread-safe. All mutable state lives in {@link DFACache},
 * which is created per-thread via {@link #createCache()}.</p>
 *
 * <p>The DFA lazily constructs states via powerset construction: each DFA state
 * is a set of simultaneously active NFA states. Transitions are computed
 * on first encounter and cached in the {@link DFACache#nextState transition table}.</p>
 *
 * <p>Patterns containing look-assertions ({@code ^}, {@code $}, {@code \b}, etc.)
 * are not supported. Use {@link #create} which returns {@code null} for such patterns.</p>
 */
public final class LazyDFA {

    private static final int DEFAULT_CACHE_CAPACITY = 2 * 1024 * 1024; // 2MB
    private static final int MIN_CLEAR_COUNT = 3;
    private static final int MIN_CHARS_PER_STATE = 10;

    private final NFA nfa;
    private final CharClasses charClasses;

    private LazyDFA(NFA nfa, CharClasses charClasses) {
        this.nfa = nfa;
        this.charClasses = charClasses;
    }

    /**
     * Creates a LazyDFA for the given NFA, or returns {@code null} if the NFA
     * contains look-assertion states which the lazy DFA cannot handle.
     */
    public static LazyDFA create(NFA nfa, CharClasses charClasses) {
        if (hasLookStates(nfa)) return null;
        return new LazyDFA(nfa, charClasses);
    }

    /** Creates a per-search cache. */
    public DFACache createCache() {
        return new DFACache(charClasses, DEFAULT_CACHE_CAPACITY, nfa.stateCount());
    }

    /**
     * Forward search for the end position of the leftmost-first match.
     *
     * @param input the search input (haystack + bounds + anchored flag)
     * @param cache per-search mutable state
     * @return Match(endPos), NoMatch, or GaveUp(offset)
     */
    public SearchResult searchFwd(Input input, DFACache cache) {
        char[] haystack = input.haystack();
        int pos = input.start();
        int end = input.end();
        int stride = charClasses.stride();
        int dead = DFACache.dead(stride);
        int quit = DFACache.quit(stride);

        int sid = getOrComputeStartState(input, cache);
        if (sid == dead) return new SearchResult.NoMatch();
        if (sid == quit) return new SearchResult.GaveUp(pos);

        int lastMatchEnd = -1;

        // Check if start state is a match
        if (sid < 0) {
            lastMatchEnd = pos;
            sid = sid & 0x7FFF_FFFF;
        }

        while (pos < end) {
            int classId = charClasses.classify(haystack[pos]);
            int nextSid = cache.nextState(sid, classId);

            if (nextSid > quit) {
                // Fast path: normal cached transition, non-special, non-match
                sid = nextSid;
                pos++;
                cache.charsSearched++;
                continue;
            }

            if (nextSid < 0) {
                // Match state (high bit set)
                lastMatchEnd = pos + 1;
                sid = nextSid & 0x7FFF_FFFF;
                pos++;
                cache.charsSearched++;
                continue;
            }

            // Slow path: UNKNOWN, DEAD, or QUIT
            if (nextSid == DFACache.UNKNOWN) {
                nextSid = computeNextState(cache, sid, classId);
                if (nextSid == quit) return new SearchResult.GaveUp(pos);
                cache.setTransition(sid, classId, nextSid);
                sid = nextSid;
                if (sid < 0) {
                    lastMatchEnd = pos + 1;
                    sid = sid & 0x7FFF_FFFF;
                }
                pos++;
                cache.charsSearched++;
                continue;
            }
            if (nextSid == dead) break;
            // nextSid == quit
            return new SearchResult.GaveUp(pos);
        }

        // EOI transition
        int eoiClass = charClasses.eoiClass();
        int eoiSid = cache.nextState(sid, eoiClass);
        if (eoiSid == DFACache.UNKNOWN) {
            eoiSid = computeNextState(cache, sid, eoiClass);
            if (eoiSid != DFACache.quit(stride)) {
                cache.setTransition(sid, eoiClass, eoiSid);
            }
        }
        if (eoiSid < 0) {
            lastMatchEnd = end;
        }

        if (lastMatchEnd >= 0) return new SearchResult.Match(lastMatchEnd);
        return new SearchResult.NoMatch();
    }

    // -- Internal: start state --

    private int getOrComputeStartState(Input input, DFACache cache) {
        if (input.isAnchored()) {
            if (cache.startAnchored == DFACache.UNKNOWN) {
                cache.startAnchored = computeStartState(nfa.startAnchored(), cache);
            }
            return cache.startAnchored;
        } else {
            if (cache.startUnanchored == DFACache.UNKNOWN) {
                cache.startUnanchored = computeStartState(nfa.startUnanchored(), cache);
            }
            return cache.startUnanchored;
        }
    }

    private int computeStartState(int nfaStartId, DFACache cache) {
        // Compute epsilon closure of the NFA start state
        cache.nfaStateSet.clear();
        boolean isMatch = epsilonClosure(cache, nfaStartId);
        int[] nfaStates = collectSorted(cache);
        if (nfaStates.length == 0) return DFACache.dead(charClasses.stride());

        StateContent content = new StateContent(nfaStates, isMatch);
        return allocateOrGiveUp(cache, content);
    }

    // -- Internal: determinization --

    /**
     * Compute the next DFA state from source state on the given class ID.
     * May clear the cache if full.
     */
    private int computeNextState(DFACache cache, int sourceSid, int classId) {
        int rawSourceId = sourceSid & 0x7FFF_FFFF;
        StateContent sourceContent = cache.getState(rawSourceId);

        // For each NFA state in the source, follow transitions for this class
        cache.nfaStateSet.clear();
        boolean isMatch = false;

        if (classId == charClasses.eoiClass()) {
            // EOI = "no more input". No char transitions to follow.
            // Match detection at EOI is already handled: if the current DFA state
            // contains an NFA Match state, it has the MATCH flag set, and the
            // search loop recorded lastMatchEnd when entering this state.
            // Patterns that require end-of-input (^ $ \b) are excluded via
            // look-assertion bailout, so no look-around check is needed here.
            return DFACache.dead(charClasses.stride());
        }

        for (int nfaStateId : sourceContent.nfaStates()) {
            State state = nfa.state(nfaStateId);
            switch (state) {
                case State.CharRange cr -> {
                    if (charInRange(classId, cr.start(), cr.end())) {
                        isMatch |= epsilonClosure(cache, cr.next());
                    }
                }
                case State.Sparse sp -> {
                    for (Transition t : sp.transitions()) {
                        if (charInRange(classId, t.start(), t.end())) {
                            isMatch |= epsilonClosure(cache, t.next());
                            break; // Sparse transitions are ordered, first match wins
                        }
                    }
                }
                default -> {
                    // Non-char-consuming states (Union, Capture, Match, Fail)
                    // shouldn't be directly in the NFA state set — they're resolved
                    // during epsilon closure. But if they are, they have no char transitions.
                }
            }
        }

        int[] nfaStates = collectSorted(cache);
        if (nfaStates.length == 0) return DFACache.dead(charClasses.stride());

        StateContent content = new StateContent(nfaStates, isMatch);
        return allocateOrGiveUp(cache, content);
    }

    /**
     * Check if the given equivalence class overlaps with [rangeStart, rangeEnd].
     *
     * <p>Equivalence classes are contiguous intervals between NFA range boundary
     * points. The builder assigns class IDs in monotonically increasing order
     * across the char space (class 0 covers [0, firstBoundary), class 1 covers
     * [firstBoundary, secondBoundary), etc.). A class either fully overlaps
     * or doesn't overlap with any NFA transition range.</p>
     *
     * <p>Since IDs are monotonic: {@code classify(rangeStart) <= classId <= classify(rangeEnd)}
     * correctly tests whether classId covers any part of [rangeStart, rangeEnd].</p>
     */
    private boolean charInRange(int classId, int rangeStart, int rangeEnd) {
        if (rangeStart > 0xFFFF || rangeEnd > 0xFFFF) return false;
        return classId >= charClasses.classify((char) rangeStart)
                && classId <= charClasses.classify((char) rangeEnd);
    }

    private int allocateOrGiveUp(DFACache cache, StateContent content) {
        if (cache.isFull()) {
            cache.clear(content);
            if (shouldGiveUp(cache)) {
                return DFACache.quit(charClasses.stride());
            }
        }
        return cache.allocateState(content);
    }

    private boolean shouldGiveUp(DFACache cache) {
        if (cache.clearCount() < MIN_CLEAR_COUNT) return false;
        // Guard: statesCreated is 0 right after a clear (before any new states).
        // Use multiplication form to avoid division by zero:
        // charsSearched / statesCreated < threshold  ⟺  charsSearched < threshold * statesCreated
        return cache.charsSearched < (long) MIN_CHARS_PER_STATE * cache.statesCreated;
    }

    // -- Internal: epsilon closure --

    /**
     * Compute epsilon closure starting from the given NFA state.
     * Adds all char-consuming states reachable via epsilon transitions
     * to {@code cache.nfaStateSet}.
     *
     * @return true if a Match state was reached
     */
    private boolean epsilonClosure(DFACache cache, int startStateId) {
        boolean isMatch = false;
        // Stack-based DFS (stack may grow if needed)
        int[] stack = cache.closureStack;
        int stackTop = 0;
        stack[stackTop++] = startStateId;

        while (stackTop > 0) {
            int sid = stack[--stackTop];
            if (!cache.nfaStateSet.insert(sid)) continue; // already visited

            State state = nfa.state(sid);
            switch (state) {
                case State.CharRange ignored -> { /* char-consuming, keep in set */ }
                case State.Sparse ignored -> { /* char-consuming, keep in set */ }
                case State.Match ignored -> isMatch = true;
                case State.Union u -> {
                    // Push in reverse order so highest-priority is processed first
                    for (int i = u.alternates().length - 1; i >= 0; i--) {
                        stack = ensureStackCapacity(stack, stackTop);
                        stack[stackTop++] = u.alternates()[i];
                    }
                }
                case State.BinaryUnion bu -> {
                    stack = ensureStackCapacity(stack, stackTop + 1);
                    stack[stackTop++] = bu.alt2();
                    stack[stackTop++] = bu.alt1();
                }
                case State.Capture cap -> {
                    // Transparent — follow through (no captures in DFA)
                    stack = ensureStackCapacity(stack, stackTop);
                    stack[stackTop++] = cap.next();
                }
                case State.Look ignored -> {
                    // Should not happen — patterns with Look bailed out.
                    // Defensive: skip this path (don't follow).
                }
                case State.Fail ignored -> { /* dead end */ }
            }
        }

        // Save possibly-grown stack back to cache
        cache.closureStack = stack;
        return isMatch;
    }

    private static int[] ensureStackCapacity(int[] stack, int needed) {
        if (needed < stack.length) return stack;
        return java.util.Arrays.copyOf(stack, stack.length * 2);
    }

    /** Collect the NFA state set into a sorted array. */
    private int[] collectSorted(DFACache cache) {
        int size = cache.nfaStateSet.size();
        int[] result = new int[size];
        for (int i = 0; i < size; i++) {
            result[i] = cache.nfaStateSet.get(i);
        }
        Arrays.sort(result);
        return result;
    }

    /** Check if any NFA state is a Look state. */
    private static boolean hasLookStates(NFA nfa) {
        for (int i = 0; i < nfa.stateCount(); i++) {
            if (nfa.state(i) instanceof State.Look) return true;
        }
        return false;
    }
}
```

**Important implementation notes for the implementer:**

1. The `charInRange` method relies on `CharClassBuilder` assigning class IDs in monotonically increasing order across the char space. This invariant means `classify(rangeStart) <= classId <= classify(rangeEnd)` correctly tests overlap. If the builder is ever refactored to use non-monotonic IDs, this method must be updated.

2. The epsilon closure does NOT need `RestoreCapture` frames like PikeVM — the lazy DFA doesn't track captures. It just finds reachable char-consuming and Match states.

3. The closure stack starts at `nfaStateCount * 2` but grows dynamically via `ensureStackCapacity` if needed. This handles deeply nested Union states safely.

4. The `computeNextState` method returns DEAD for EOI — match detection at EOI happens via the source state's MATCH flag, already checked in the search loop. Patterns requiring EOI semantics (`$`, `\b`) are excluded by the look-assertion bailout.

5. The give-up heuristic uses multiplication form (`charsSearched < threshold * statesCreated`) instead of division to avoid divide-by-zero when `statesCreated` is 0 right after a cache clear.
